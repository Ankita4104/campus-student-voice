# 🎓 Campus Student Voice

A student-focused web platform built to give students a voice by allowing them to submit issues, concerns, and opinions through a clean and interactive interface.

This project uses frontend development along with backend database integration to dynamically store and display user-submitted content.

## ✨ Features

* Responsive modern UI
* Student issue submission system
* Dynamic data rendering
* Firebase backend integration
* Real-time data storage
* Clean user-friendly interface
* Mobile responsive design
* Smooth frontend interactions

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Firebase

## 🚀 Live Website

https://campus-student-voice.netlify.app/

## 📌 Project Purpose

Campus Student Voice was created to provide students with a platform where they can openly share concerns, issues, and feedback in a simple digital space.

The project focuses on combining frontend design with backend functionality to create a practical student-oriented solution.
